# Titanic Survival Analysis

## 📌 Project Overview
This project analyzes the Titanic dataset to uncover key factors influencing passenger survival.

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Identify patterns affecting survival
- Generate actionable insights

## 🛠 Tools Used
- Python
- Pandas (Data Analysis)
- Matplotlib (Visualization)


## 📊 Key Analysis Performed
- Feature Engineering (AgeGroup, SibSpGroup, FareGroup)
- Survival analysis by gender, class, age, and fare
- Distribution analysis (Age, Fare)
- Multi-variable grouping

## 🔍 Key Insights
- Male passengers had significantly higher survival rates due to evacuation priority
- 3rd-class passengers had better survival outcomes.
- Higher fare correlated with higher survival probability
- Smaller family groups had higher survival rates

## 🧠 Conclusion
Survival was strongly influenced by gender, class, and wealth, showing clear inequality in outcomes rather than random chance.
